## 📊 算法原理

### 模块 1：参数化车身建模
- **输入**：22 维 `CarParams`（车长 L / 车宽 W / 车高 H / 轴距 + 18 项造型参数）
- **算法**：沿车长方向取 N 个截面（默认 48）→ 每截面"宽+高+形状"3 个函数 → 三角网格
- **8 部件**：车壳 / 玻璃（前+后+天窗+侧窗）/ 4 轮 / 大灯 / 尾灯 / 格栅 / 后视镜 / 门缝
- **输出**：8 个独立 trimesh，可单独操作或合并导出 GLB/STL/OBJ/PLY

### 模块 2：曲面质量评估
- **G0**：位置连续（默认全部三角面满足）
- **G1**：切线连续（跨边法向夹角 < 5°）
- **G2**：曲率连续（跨边法向夹角 < 2°）
- **反射线评分**：曲率均匀度 × 0.5 + 曲率平滑度 × 0.5
- **综合等级**：
  - A 级: G2 比率 > 85% 且反射线 > 0.7
  - B 级: G2 比率 > 70% 且反射线 > 0.5
  - C 级: G2 比率 > 50%
  - D 级: 其他

### 模块 3：AI 模拟退火优化
- **目标函数**：法向跳变平方和（越小越光顺）
- **算法**：
  1. 初始温度 T = 1.0
  2. 每步随机扰动一个内部点（加高斯噪声 × T × lr）
  3. **Metropolis 接受准则**：目标更优直接接受；目标更差按概率 exp(-Δ/T) 接受
  4. 降温：T *= cooling（默认 0.97）
  5. 终止：T < min_temp 或达到 max_iter

### 模块 4：视频脚本生成
- **3 套内置模板**：
  - car_promotion：7 镜 90s（汽车产品宣传片）
  - tech_demo：5 镜 90s（技术演示片）
  - minimal_showcase：2 镜 90s（极简展示）
- **支持自定义分镜**（覆盖模板）+ **自动时长缩放**

### 模块 5：视频脚本展示
- **Markdown**：表格化镜头信息 + 数据对比表 + 配色色卡
- **HTML**：响应式深色布局 + 卡片式分镜 + 配色预览 + 数据高亮
