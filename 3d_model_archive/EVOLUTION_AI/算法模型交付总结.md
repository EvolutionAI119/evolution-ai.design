## 🔬 算法原理

### 5.1 NURBS曲面数学基础

**B样条基函数**（De Boor-Cox递推公式）：
```
N_i^0(u) = 1, if u ∈ [knots[i], knots[i+1])
           0, otherwise

N_i^k(u) = (u - knots[i])/(knots[i+k] - knots[i]) * N_i^(k-1)(u) 
         + (knots[i+k+1] - u)/(knots[i+k+1] - knots[i+1]) * N_i+1^(k-1)(u)
```

**NURBS曲面评估公式**：
```
S(u, v) = Σ(i=0 to nu-1) Σ(j=0 to nv-1) w_ij * P_ij * N_i^p(u) * N_j^q(v)
          / Σ(i=0 to nu-1) Σ(j=0 to nv-1) w_ij * N_i^p(u) * N_j^q(v)
```

### 5.2 硬点驱动设计

**14个硬点参数** → **34个车身部件** → **G2连续曲面**

| 参数分类 | 参数 | 影响 |
|----------|------|------|
| 尺寸参数 | L, W, H, WB, FO, RO | 车身基本尺寸 |
| 姿态参数 | GC, WR, TW | 车身姿态和稳定性 |
| 造型参数 | AA, CA, WL, WBulge | 车身造型风格 |

### 5.3 G2连续性验证

**验证条件**：
1. 相邻曲面边界点重合（G0）
2. 边界处法向量方向一致（G1）
3. 边界处曲率值相等（G2）

**质量评估指标**：
- 高斯曲率：-1.0 ~ 1.0
- 平均曲率：-0.5 ~ 0.5
- 曲率梳偏差：< 0.01
- 法向量偏差：< 2°
