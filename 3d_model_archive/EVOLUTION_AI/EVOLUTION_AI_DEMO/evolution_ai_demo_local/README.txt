一、这是什么
------------
EVOLUTION AI 站点 W2-D1 阶段的本地预览包。
- 22 维汽车造型参数 + Three.js 真实 3D 渲染
- 5 个滑块分组 / 3 套预设 / 5 个视角 / 自动旋转
- 4 个统计卡片 + 部件详情折叠表
- 全 5 个页面：首页 / 项目 / 质量 / 分镜 / 造型设计

二、文件结构
------------
交付_evolution_ai_demo_w2d1/
  start.bat               双击启动本地服务（UTF-8 BOM 编码）
  README.txt              本文件
  index.html              入口 HTML（已本地化 CDN）
  favicon.svg             站点图标
  vendor/                 第三方库本地文件（vue/element-plus/axios 等）
  assets/                 vite 编译产物（js + css）

三、启动方式
------------
1. 双击 start.bat
2. 浏览器自动打开 http://localhost:8080/
3. 顶部菜单点 "造型设计" 进入 3D 设计师

要求：
  - Windows 系统
  - Python 3.x 已安装（用于 python -m http.server）
  - 端口 8080 未被占用（被占用会自动 kill）

四、3D 设计师怎么玩
-------------------
进入 /designer 页面后：

【左侧控制面板 - 22 维参数】
  - 基础尺寸 (4)：L / W / H / 轴距
  - 比例姿态 (4)：车头长 / 座舱长 / 尾厢长 / 离地间隙
  - 曲面特征 (8)：前盖角 / 车顶弧 / 风窗倾角 / 后窗倾角 / 翼子板 / 腰线 / 肩线 / 整体弧
  - 玻璃与轮 (4)：玻璃深度 / 轮毂半径 / 轮毂宽度 / 辐条数
  - 灯 (2)：前灯宽 / 前灯高

【拖动滑块】
  - 自动触发 /api/v1/car/build 请求
  - 400ms 防抖
  - 加载新 GLB 模型 + 更新统计

【视角控制】
  - 鼠标拖动旋转
  - 滚轮缩放
  - 右键平移
  - 视角预设按钮：正 / 侧 / 后 / 45 / 俯
  - 自动旋转开关

【预设车型】
  - 运动型：低矮、宽体、激进角度
  - 豪华型：长轴距、优雅曲面
  - SUV：高离地间隙、方正比例

五、后端联调（真实 NURBS 渲染）
-------------------------------
3D 设计师默认调的是前端占位，**真实后端联调需要起 backend**：

1. 主人电脑上有 start_backend.bat
2. 双击它启动后端（端口 8000）
3. 前端 vite proxy 自动转发 /api → http://127.0.0.1:8000
4. 拖动滑块会调真实后端，返回 GLB

如果后端没起：
  - 页面能正常打开
  - 拖动滑块会显示 "无法连接到后端" 错误
  - 5 个视角预设仍可看 3D 占位

六、注意事项
------------
- 本包是 build 后的纯静态产物（dist/），不是源码
- Python 必须装好；如提示找不到 python，请先装 Python 3
- 不要直接双击 index.html（file:// 协议会被 CORS 拦）
- 浏览器推荐 Chrome / Edge（WebGL 兼容性最好）
