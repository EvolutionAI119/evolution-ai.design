## 1. 概述

### 1.1 项目背景

EVOLUTION AI 是一套基于人工智能的汽车A级曲面开发系统，采用**NURBS驱动的硬点设计范式**（NURBS-Driven Hardpoint Design），通过定义核心硬点参数，实时生成符合物理约束、国标要求且满足G2曲率连续的车身曲面。

### 1.2 设计理念

- **从"画车"到"造车"**：摒弃传统数学曲线绘制方式，采用物理硬点约束驱动车身形态
- **从"三角网格"到"NURBS曲面"**：采用工业级NURBS曲面技术，实现真正的G2连续性
- **数据驱动决策**：真实市场标杆车型参数作为设计基准
- **AI赋能创意**：自然语言输入 + 智能参数推荐 + 趋势分析

## 2. 核心方法论：NURBS驱动的硬点设计

### 2.1 范式定义

**NURBS驱动的硬点设计**是一种以车辆核心几何硬点为设计起点，通过NURBS曲面引擎生成G2连续车身曲面的汽车造型开发方法。

| 硬点类别 | 包含点位 | 设计意义 |
|----------|----------|----------|
| 车轮硬点 | 前轮心、后轮心、轮径、轮距 | 决定车身姿态和通过性 |
| 乘员硬点 | H点（臀点）、人眼点 | 决定乘员舱空间和视野 |
| 车身硬点 | A柱底、A柱顶、C柱底、C柱顶 | 决定车身侧面轮廓 |
| 比例硬点 | 腰线高度、发动机盖高度 | 决定车身比例关系 |

### 2.2 硬点推导公式

```python
# 轮心坐标推导
fwx = FO + GC                    # 前轮心X坐标
rwx = L - RO                     # 后轮心X坐标
wcy = GC                         # 轮心Y坐标（离地高度）
fwz = TW / 2                     # 轮心外侧偏移

# 验证约束
assert rwx - fwx == WB, "轴距约束验证失败"

# A柱推导
aBaseX = fwx + 100                # A柱底在前轴后100mm
aTopY = H * 0.92                 # A柱顶高度
aTopX = aBaseX + (aTopY - waistY) / tan(AA°)

# C柱推导
cBaseX = rwx + 300               # C柱底在后轴后300mm
cTopX = cBaseX - 500             # C柱顶在C柱底前500mm
cTopY = aTopY - 30
```

### 2.3 几何约束

| 约束条件 | 公式 | 意义 |
|----------|------|------|
| 车身总长 | L = FO + WB + RO | 前后悬+轴距=总长 |
| 轮心高度 | WC = GC + WR | 离地间隙+轮径=轮心高 |
| A柱倾角 | tan(AA) = ΔY / ΔX | 腰线高度差 / 水平距离差 |
| 轮距约束 | TW < W | 轮距必须小于车宽 |
| 轮径约束 | WR * 2 < H * 0.6 | 车轮直径不超过车高60% |

## 3. 五阶段开发流程

### Phase 1: 硬点定义

**目标**：建立车辆核心硬点，定义物理约束边界

**输入**：轮心位置、H点、人眼点、A/C柱角度

**输出**：硬点坐标矩阵

**核心原理**：
- 轮心位置决定车身姿态和比例
- H点决定乘员舱空间布局
- 人眼点决定视野设计
- 前轮心X = FO + GC
- 后轮心X = L - RO

**关联参数**：FO、RO、WB、GC、WR、TW、AA、CA

---

### Phase 2: NURBS控制点布局

**目标**：构建车身各部件的NURBS控制点网格

**输入**：硬点坐标、设计意图、NURBS模板配置

**输出**：各部件NURBS曲面控制点矩阵

**核心原理**：
- 根据部件类型选择对应的NURBS模板（degree_u/v, num_u/v）
- 使用三角函数和几何函数生成控制点坐标
- 确保相邻曲面边界控制点对齐（G0连续）
- 确保边界处法向量方向一致（G1连续）

**NURBS模板配置示例**：

| 部件 | degree_u | degree_v | num_u | num_v |
|------|----------|----------|-------|-------|
| 发动机盖 | 5 | 3 | 12 | 8 |
| 前风挡 | 3 | 3 | 8 | 6 |
| 车顶 | 5 | 3 | 10 | 8 |
| 后风挡 | 3 | 3 | 8 | 6 |
| 行李箱盖 | 3 | 3 | 8 | 6 |
| 翼子板 | 3 | 3 | 6 | 6 |
| 车门 | 3 | 3 | 8 | 8 |
| 保险杠 | 3 | 3 | 6 | 8 |

**控制点布局算法**：
```python
for i in range(num_u):
    u = i / (num_u - 1)
    row = []
    for j in range(num_v):
        v = j / (num_v - 1)
        
        # X坐标：线性分布
        x = cx_start + u * length
        
        # Y坐标：三角函数控制纵向和横向曲率
        y = cy_base + height * sin(u * π) * cos(v * π)
        
        # Z坐标：线性分布 + Tumblehome内倾
        z = (v - 0.5) * width * (1 - u * tumblehome_factor)
        
        row.append(ControlPoint(x, y, z, 1.0))
    control_points.append(row)
```

### Phase 3: NURBS曲面生成

**目标**：基于控制点和节点矢量生成G2连续曲面

**输入**：控制点矩阵、degree_u/v、节点矢量

**输出**：NURBSSurface对象

**核心原理**：
- **B样条基函数**：使用De Boor-Cox递推公式计算
- **NURBS曲面评估**：加权控制点 × 双变量B样条基函数
- **节点矢量**：自动生成均匀节点矢量，确保曲面平滑
- **权重控制**：默认权重1.0，可调整以实现局部形状控制

**NURBS曲面评估公式**：
```
S(u, v) = Σ(i=0 to nu-1) Σ(j=0 to nv-1) w_ij * P_ij * N_i^p(u) * N_j^q(v)
          / Σ(i=0 to nu-1) Σ(j=0 to nv-1) w_ij * N_i^p(u) * N_j^q(v)
```

---

### Phase 4: 曲面质量评估

**目标**：验证曲面质量，确保G2连续性

**输入**：NURBS曲面、采样参数

**输出**：质量评估报告

**核心原理**：
- **法向量计算**：通过交叉偏导数计算曲面法向量
- **曲率计算**：通过二阶导数计算高斯曲率和平均曲率
- **G2连续性验证**：检查相邻曲面边界处曲率值是否相等

**连续性验证条件**：
1. 相邻曲面边界点重合（G0）
2. 边界处法向量方向一致（G1）
3. 边界处曲率值相等（G2）

---

### Phase 5: 车身网格生成与导出

**目标**：将NURBS曲面转换为可渲染的三角网格并导出

**输入**：NURBS曲面、采样密度

**输出**：GLB/STL/OBJ/STEP格式的3D模型

**核心原理**：
- **NURBS曲面采样**：在u/v参数域均匀采样生成点云
- **三角面构建**：将四边形网格分解为两个三角形
- **轮拱切口处理**：检测并切除车轮干涉区域的点
- **格式导出**：支持GLB（Web）、STL（3D打印）、OBJ（通用）、STEP（工程）

## 4. NURBS曲面引擎

### 4.1 引擎架构

```
┌─────────────────────────────────────────────────────────────┐
│                     NURBS引擎层                             │
├─────────────────────────────────────────────────────────────┤
│  ControlPoint: 控制点类 (x, y, z, weight)                   │
│  KnotVector: 节点矢量类 (均匀/非均匀)                        │
│  NURBSCurve: NURBS曲线类 (B样条基函数计算)                  │
│  NURBSSurface: NURBS曲面类 (双变量B样条评估)                │
│  SurfaceModifier: 曲面修改工具 (平移/缩放/旋转/偏移)         │
│  CurveModifier: 曲线修改工具 (平移/缩放)                     │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 B样条基函数

**De Boor-Cox递推公式**：
```
N_i^0(u) = 1, if u ∈ [knots[i], knots[i+1])
           0, otherwise

N_i^k(u) = (u - knots[i])/(knots[i+k] - knots[i]) * N_i^(k-1)(u) 
         + (knots[i+k+1] - u)/(knots[i+k+1] - knots[i+1]) * N_i+1^(k-1)(u)
```

### 4.3 节点矢量生成

```python
@staticmethod
def _create_uniform_knot_vector(num_control: int, degree: int) -> KnotVector:
    n = num_control
    p = degree
    num_knots = n + p + 2
    
    knots = []
    # 前p+1个节点为0
    for i in range(p + 1):
        knots.append(0.0)
    
    # 中间节点均匀分布
    num_internal = n - p
    if num_internal > 0:
        for i in range(1, num_internal + 1):
            knots.append(i / (num_internal + 1))
    
    # 后p+1个节点为1
    for i in range(p + 1):
        knots.append(1.0)
    
    return KnotVector(knots)
```

### 4.4 法向量与曲率计算

```python
def evaluate_normal(self, u: float, v: float) -> np.ndarray:
    eps = 0.001
    p1 = self.evaluate_point(u + eps, v)
    p2 = self.evaluate_point(u - eps, v)
    p3 = self.evaluate_point(u, v + eps)
    p4 = self.evaluate_point(u, v - eps)
    
    du = p1 - p2  # u方向切向量
    dv = p3 - p4  # v方向切向量
    
    normal = np.cross(du, dv)  # 叉积计算法向量
    return normalize(normal)

def evaluate_curvature(self, u: float, v: float) -> Dict[str, float]:
    eps = 0.001
    p = self.evaluate_point(u, v)
    puu = (self.evaluate_point(u+eps,v) - 2*p + self.evaluate_point(u-eps,v)) / eps²
    pvv = (self.evaluate_point(u,v+eps) - 2*p + self.evaluate_point(u,v-eps)) / eps²
    
    normal = self.evaluate_normal(u, v)
    
    return {
        'gaussian_curvature': dot(puu, normal) * dot(pvv, normal),
        'mean_curvature': (dot(puu, normal) + dot(pvv, normal)) / 2,
        'principal_curvature_1': dot(puu, normal),
        'principal_curvature_2': dot(pvv, normal)
    }
```

## 5. 车身部件NURBS生成算法

### 5.1 发动机盖

**模板配置**：degree_u=5, degree_v=3, num_u=12, num_v=8

**控制点布局算法**：
```python
for i in range(num_u):
    u = i / (num_u - 1)
    for j in range(num_v):
        v = j / (num_v - 1)
        
        x = cx_start + u * length
        y = cy_base + height * sin(u * π) * cos(v * π) + u * tan(angle) * length * 0.3
        z = (v - 0.5) * width
```

**设计原理**：
- `sin(u * π)`: 控制发动机盖纵向弧度（前低后高）
- `cos(v * π)`: 控制发动机盖横向弧度（中间高两侧低）
- `tan(angle)`: 控制发动机盖倾斜角度

### 5.2 前风挡玻璃

**模板配置**：degree_u=3, degree_v=3, num_u=8, num_v=6

**控制点布局算法**：

```python
angle = radians(90 - windshield_angle)

for i in range(num_u):

        x = cx_base - u * height * sin(angle)
        y = cy_bottom + u * height * cos(angle)

```

### 5.3 车顶

## 6. 14个硬点参数详解

### 6.1 参数总览

| 参数代码 | 参数名称 | 单位 | 合理范围 | 影响描述 |
|----------|----------|------|----------|----------|
| L | 车长 | m | 2.5-6.0 | 整体比例感，越长越豪华 |
| W | 车宽 | m | 1.4-2.2 | 稳定性和内部空间 |
| H | 车高 | m | 1.0-2.8 | 重心位置和视觉重心 |
| WB | 轴距 | m | 1.8-4.0 | 决定前后悬分配 |
| FO | 前悬 | m | 0.3-1.2 | 前鼻长度，越短越运动 |
| RO | 后悬 | m | 0.3-1.5 | 车尾长度，影响尾部造型 |
| GC | 离地间隙 | m | 0.08-0.30 | 通过性和姿态高度 |
| WR | 轮径 | m | 0.24-0.45 | 车轮尺寸和视觉比例 |
| TW | 轮距 | m | 1.2-1.8 | 稳定性和侧面比例 |
| AA | A柱角 | ° | 15-40 | 前风挡倾斜角度 |
| CA | C柱角 | ° | 25-65 | 后风挡倾斜角度 |
| WL | 腰线高 | m | 0.4-1.0 | 侧面黄金分割线 |
| WBulge | 轮拱凸出 | m | 0.0-0.10 | 轮拱肌肉感 |

### 6.2 参数分类

#### 尺寸参数（6个）
- **L**：车身总长，决定车型级别（微型/紧凑/中型/大型）
- **W**：车身总宽，影响内部空间和行驶稳定性
- **H**：车身总高，决定重心和视觉风格（低趴/高挺）
- **WB**：轴距，决定前后排空间和操控性
- **FO**：前悬，影响前鼻造型和接近角
- **RO**：后悬，影响尾部造型和离去角

#### 姿态参数（3个）
- **GC**：离地间隙，决定通过性和姿态高度
- **WR**：轮径，影响视觉比例和行驶舒适性
- **TW**：轮距，影响横向稳定性

#### 造型参数（4个）
- **AA**：A柱角，影响风挡角度和视野
- **CA**：C柱角，影响尾部流线和后排头部空间
- **WL**：腰线高，决定侧面比例和视觉重心
- **WBulge**：轮拱凸出，影响肌肉感和运动风格

### 6.3 参数比例关系

| 比例 | 计算公式 | 合理范围 | 设计意义 |
|------|----------|----------|----------|
| 轴距/车长 | WB/L | 0.55-0.70 | 越大越豪华，越小越运动 |
| 前悬/总长 | FO/L | 0.15-0.20 | 越小越运动 |

## 7. G2连续性验证体系

### 7.1 连续性等级

| 连续性等级 | 含义 | 数学条件 | 视觉表现 |
|------------|------|----------|----------|
| G0（位置连续） | 曲面边界相连 | 边界点相同 | 无缺口，但可能有折痕 |
| G1（切线连续） | 边界切线方向一致 | 法向量连续 | 无折痕，反射光连续 |
| G2（曲率连续） | 边界曲率相同 | 曲率向量连续 | 反射光平滑过渡 |
| G3（曲率变化率连续） | 曲率变化率一致 | 曲率导数连续 | 极高质量的A级曲面 |

### 7.2 G2连续性验证流程

```
1. 生成初始NURBS曲面
    ↓
2. 边界点检查 → 确保G0连续（边界点重合）
    ↓
3. 法向量检查 → 确保G1连续（边界法向量方向一致）
    ↓
4. 曲率检查 → 确保G2连续（边界曲率值相等）
    ↓
5. 斑马纹检查 → 验证曲面光顺性
    ↓
6. 高光检查 → 验证反射连续性
    ↓
7. 完成G2连续曲面
```

### 7.3 质量评估指标

| 指标 | 计算公式 | 合格标准 |
|------|----------|----------|
| 高斯曲率 | K = k1 × k2 | -1.0 ~ 1.0 |
| 平均曲率 | H = (k1 + k2) / 2 | -0.5 ~ 0.5 |
| 曲率梳偏差 | max(|k_boundary - k_internal|) | < 0.01 |
| 法向量偏差 | angle(n_boundary, n_internal) | < 2° |

```python
def checkGB1589(HP):
```
