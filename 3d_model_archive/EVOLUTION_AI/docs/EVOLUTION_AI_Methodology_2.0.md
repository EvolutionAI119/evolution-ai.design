## 1. 概述

### 1.1 项目背景

EVOLUTION AI 是一套基于人工智能的汽车A级曲面开发系统，采用**硬点驱动设计范式**（Hardpoint-Driven Design），通过定义核心硬点参数，实时生成符合物理约束和国标要求的车身曲面。

### 1.2 设计理念

- **从"画车"到"造车"**：摒弃传统数学曲线绘制方式，采用物理硬点约束驱动车身形态
- **数据驱动决策**：真实市场标杆车型参数作为设计基准
- **AI赋能创意**：自然语言输入 + 智能参数推荐 + 趋势分析

## 2. 核心方法论：硬点驱动设计

### 2.1 范式定义

**硬点驱动设计**是一种以车辆核心几何硬点为设计起点的汽车造型开发方法。硬点是决定车辆基本形态和性能的关键几何控制点，包括：

| 硬点类别 | 包含点位 | 设计意义 |
|----------|----------|----------|
| 车轮硬点 | 前轮心、后轮心、轮径、轮距 | 决定车身姿态和通过性 |
| 乘员硬点 | H点（臀点）、人眼点 | 决定乘员舱空间和视野 |
| 车身硬点 | A柱底、A柱顶、C柱底、C柱顶 | 决定车身侧面轮廓 |
| 比例硬点 | 腰线高度、发动机盖高度 | 决定车身比例关系 |

### 2.2 硬点推导公式

```javascript
// 轮心位置推导
var fwx = HP.FO + HP.WR + 0.08;        // 前轮心X坐标
var rwx = HP.L - HP.RO - HP.WR - 0.08; // 后轮心X坐标
var wcy = HP.GC + HP.WR;               // 轮心Y坐标（离地高度+轮径）
var fwz = HP.TW / 2 + HP.WR * 0.35;    // 轮心Z坐标（半轮距+偏移）

// A柱位置推导
var aBaseX = fwx + HP.WR + 0.05;       // A柱底X坐标
var aTopY = wcy + HP.WL;               // A柱顶Y坐标（轮心高+腰线高）
var aTopX = aBaseX + (aTopY - wcy) / Math.tan(HP.AA * Math.PI / 180); // A柱顶X坐标

// C柱位置推导
var cBaseX = rwx - HP.WR - 0.05;       // C柱底X坐标
var cTopY = wcy + HP.WL * 0.9;         // C柱顶Y坐标（略低于腰线）
var cTopX = cBaseX - (cTopY - wcy) / Math.tan(HP.CA * Math.PI / 180); // C柱顶X坐标
```

### 2.3 几何约束

| 约束条件 | 公式 | 意义 |
|----------|------|------|
| 车身总长 | L = FO + WB + RO | 前后悬+轴距=总长 |
| 轮心高度 | WC = GC + WR | 离地间隙+轮径=轮心高 |
| A柱倾角 | tan(AA) = ΔY / ΔX | 腰线高度差 / 水平距离差 |
| 轮距约束 | TW < W | 轮距必须小于车宽 |
| 轮径约束 | WR * 2 < H * 0.6 | 车轮直径不超过车高60% |

## 3. 五阶段开发流程

### Phase 1: 硬点定义

**目标**：建立车辆核心硬点，定义物理约束边界

**输入**：轮心位置、H点、人眼点、A/C柱角度

**输出**：硬点坐标矩阵

**核心原理**：
- 轮心位置决定车身姿态和比例
- H点决定乘员舱空间布局
- 人眼点决定视野设计
- 前轮心X = FO + WR + 0.08（前悬+轮径+偏移）
- 后轮心X = L - RO - WR - 0.08（总长-后悬-轮径-偏移）

**关联参数**：FO、RO、WB、GC、WR、TW、AA、CA

---

### Phase 2: 三曲线系统

**目标**：构建车身骨架曲线，定义车身形态

**输入**：硬点坐标、设计意图

**输出**：侧视轮廓曲线、顶视宽度曲线、截面变形函数

**核心原理**：
- **侧视轮廓曲线**：定义车身侧面造型，从车头到车尾的Y坐标变化
- **顶视宽度曲线**：定义车身宽度变化，实现tumblehome内倾效果
- **截面变形函数**：控制垂直方向的轮廓变化，确保G1连续

**平滑插值算法**：
```javascript
function smoothstep(t) {
    return t * t * (3 - 2 * t); // t²(3-2t)，确保G1连续
}
```

**Tumblehome内倾**：乘员舱区域车顶向内倾斜，减少空气阻力并提升侧面造型流畅度

**关联参数**：L、W、H、WL、WBulge、AA、CA

---

### Phase 3: 参数化建模

**目标**：14个可调参数实时驱动车身形态变化

**输入**：用户参数调整、AI推荐参数

**输出**：更新后的车身曲面

**核心原理**：
- 车身总长L决定整体比例感
- 轴距WB决定前后悬分配比例
- 前悬FO决定前鼻长度和攻击性
- 后悬RO决定车尾长度和姿态
- 车高H决定重心位置和视觉重心
- 腰线WL决定侧面黄金分割比例

**参数驱动流程**：
```
用户调整参数 → 参数校验 → 硬点重新推导 → 曲线重新生成 → 曲面重新扫掠 → 渲染更新
```

**关联参数**：L、W、H、WB、FO、RO、WL

---

### Phase 4: A级曲面

**目标**：生成符合A级曲面质量要求的车身表面

**输入**：三曲线系统输出、截面数量

**输出**：G2连续的车身曲面

**核心原理**：
- **Lofted Surface截面扫掠**：50站截面 × 16层宽度，生成闭合环面
- **G2曲率连续**：确保曲率变化平滑，无尖点和折痕
- **斑马纹检查**（F5）：验证曲面光顺性
- **高光检查**（F6）：验证反射连续性
- **曲率梳检查**（F7）：验证曲率分布均匀性

**曲面质量标准**：

| 检查项 | 合格标准 |
|--------|----------|
| 位置连续(G0) | 无断点 |
| 切线连续(G1) | 无折痕 |
| 曲率连续(G2) | 曲率梳无尖峰 |
| 斑马纹 | 无交叉、无突变 |
| 高光 | 轨迹平滑、无跳跃 |

**关联参数**：WBulge、AA、CA、WR、TW

---

### Phase 5: AI辅助设计

**目标**：利用AI技术辅助创意设计和决策

**输入**：自然语言描述、设计需求

**输出**：参数方案、造型推荐、趋势分析

**核心功能**：
1. **语言参数输入**：自然语言转硬点参数
2. **智能参数推荐**：AI根据车型自动推荐最优参数组合
3. **造型趋势分析**：分析市场最新车型设计趋势
4. **概念车探索**：生成未来概念车型造型方案
5. **多方案并行对比**：同时生成多种造型方案进行对比

**AI分析算法示例**：
```javascript
function analyzeDesignInput(input) {
    var params = {};
    if (input.includes('轿跑') || input.includes('跑车')) {
        params.L = 4.85; params.H = 1.38; params.WL = 0.62; params.WBulge = 0.05;
    }
    if (input.includes('SUV')) {
        params.H = 1.70; params.GC = 0.20; params.WL = 0.82;
    }
    if (input.includes('车长') && input.match(/([\d.]+)米/)) {
        params.L = parseFloat(input.match(/([\d.]+)米/)[1]);
    }
    return params;
}
```

## 4. 三曲线系统

### 4.1 侧视轮廓曲线

**定义**：车身侧面从车头到车尾的Y坐标变化曲线

**控制点**：
- 前鼻点：Y = GC + WR + 0.05
- A柱底点：Y = GC + WR + 0.08
- A柱顶点：Y = GC + WR + WL
- 车顶中点：Y = H * 0.95
- C柱顶点：Y = GC + WR + WL * 0.9
- C柱底点：Y = GC + WR + 0.08
- 车尾点：Y = GC + WR + 0.05

**曲线方程**：采用smoothstep插值，确保G1连续

### 4.2 顶视宽度曲线

**定义**：车身宽度沿X轴的变化曲线

**控制点**：
- 车头宽度：W * 0.7（较窄，空气动力学）
- 前轮拱宽度：W * 0.95 + WBulge（最宽处）
- A柱宽度：W * 0.85（开始内倾）
- 乘员舱宽度：W * 0.75（tumblehome内倾）
- C柱宽度：W * 0.72
- 后轮拱宽度：W * 0.95 + WBulge（最宽处）
- 车尾宽度：W * 0.75

**Tumblehome内倾**：乘员舱区域宽度向内收缩，角度约3-5度

### 4.3 截面变形函数

**定义**：垂直方向（Y轴）的轮廓变形函数

**功能**：
- 控制车身侧面弧度
- 实现腰线造型
- 控制轮拱凸出
- 确保各截面之间的平滑过渡

## 5. 14个硬点参数详解

### 5.1 参数总览

| 参数代码 | 参数名称 | 单位 | 合理范围 | 影响描述 |
|----------|----------|------|----------|----------|
| L | 车长 | m | 2.5-6.0 | 整体比例感，越长越豪华 |
| W | 车宽 | m | 1.4-2.2 | 稳定性和内部空间 |
| H | 车高 | m | 1.0-2.8 | 重心位置和视觉重心 |
| WB | 轴距 | m | 1.8-4.0 | 决定前后悬分配 |
| FO | 前悬 | m | 0.3-1.2 | 前鼻长度，越短越运动 |
| RO | 后悬 | m | 0.3-1.5 | 车尾长度，影响尾部造型 |
| GC | 离地间隙 | m | 0.08-0.30 | 通过性和姿态高度 |
| WR | 轮径 | m | 0.24-0.45 | 车轮尺寸和视觉比例 |
| TW | 轮距 | m | 1.2-1.8 | 稳定性和侧面比例 |
| AA | A柱角 | ° | 15-40 | 前风挡倾斜角度 |
| CA | C柱角 | ° | 25-65 | 后风挡倾斜角度 |
| WL | 腰线高 | m | 0.4-1.0 | 侧面黄金分割线 |
| WBulge | 轮拱凸出 | m | 0.0-0.10 | 轮拱肌肉感 |

### 5.2 参数分类

#### 尺寸参数（6个）
- **L**：车身总长，决定车型级别（微型/紧凑/中型/大型）
- **W**：车身总宽，影响内部空间和行驶稳定性
- **H**：车身总高，决定重心和视觉风格（低趴/高挺）
- **WB**：轴距，决定前后排空间和操控性
- **FO**：前悬，影响前鼻造型和接近角
- **RO**：后悬，影响尾部造型和离去角

#### 姿态参数（3个）
- **GC**：离地间隙，决定通过性和姿态高度
- **WR**：轮径，影响视觉比例和行驶舒适性
- **TW**：轮距，影响横向稳定性

#### 造型参数（4个）
- **AA**：A柱角，影响风挡角度和视野
- **CA**：C柱角，影响尾部流线和后排头部空间
- **WL**：腰线高，决定侧面比例和视觉重心
- **WBulge**：轮拱凸出，影响肌肉感和运动风格

### 5.3 参数比例关系

| 比例 | 计算公式 | 合理范围 | 设计意义 |
|------|----------|----------|----------|
| 轴距/车长 | WB/L | 0.55-0.70 | 越大越豪华，越小越运动 |
| 前悬/总长 | FO/L | 0.15-0.20 | 越小越运动 |
| 后悬/总长 | RO/L | 0.20-0.25 | 越小越紧凑 |
| 高宽比 | H/W | 0.65-0.95 | 越小越低趴，越大越箱式 |
| 轮径/车高 | WR/H | 0.18-0.25 | 影响视觉平衡 |
| 轮距/车宽 | TW/W | 0.80-0.88 | 接近0.85为理想值 |

## 6. 国标约束体系

### 6.1 GB 1589-2016 乘用车限值

| 参数 | 限值 | 说明 |
|------|------|------|
| 车长(L) | ≤ 5.3m | M1类乘用车限值 |
| 车宽(W) | ≤ 2.55m | 不含后视镜 |
| 车高(H) | ≤ 4.0m | 不含行李架 |
| 离地间隙(GC) | ≥ 0.13m | 乘用车最小值 |

### 6.2 GB 7258-2017 安全技术条件

- 轴距必须保证乘员安全空间
- 前悬不得过长影响碰撞安全
- 离地间隙必须满足行驶安全

### 6.3 合规检查流程

```javascript
function checkGB1589(HP) {
    var issues = [];
    if (HP.L > 5.3) issues.push('车长超限');
    if (HP.W > 2.55) issues.push('车宽超限');
    if (HP.H > 4.0) issues.push('车高超限');
    if (HP.GC < 0.13) issues.push('离地间隙不足');
    return issues;
}
```

### 6.4 例外车型说明

部分车型因特殊定位超出国标限值，属合理例外：

| 车型 | 违规项 | 说明 |
|------|--------|------|
| 加长轿车（迈巴赫） | L=5.47m > 5.3m | 超豪华定位，属M1类高端车型 |
| 跑车/超跑 | GC=0.10-0.12m < 0.13m | 低重心设计，可升降底盘 |
| 轻客（Sprinter） | L=5.93m > 5.3m | 属N1类商用车，不受M1限值约束 |
| 皮卡 | L=5.34m > 5.3m | 属N1类，不受M1限值约束 |

## 7. A级曲面质量检查

### 7.1 检查体系

| 检查项 | 快捷键 | 检查内容 | 合格标准 |
|--------|--------|----------|----------|
| 斑马纹 | F5 | 曲面光顺性 | 条纹平滑、无交叉、无断点 |
| 高光 | F6 | 反射连续性 | 高光轨迹平滑、无突变 |
| 曲率梳 | F7 | 曲率分布 | 曲率梳均匀、无尖峰 |
| 截面检查 | F8 | 截面质量 | 截面曲线连续、无自交 |

### 7.2 连续性标准

| 连续性等级 | 含义 | 数学条件 | 视觉表现 |
|------------|------|----------|----------|
| G0（位置连续） | 曲面边界相连 | 边界点相同 | 无缺口，但可能有折痕 |
| G1（切线连续） | 边界切线方向一致 | 法向量连续 | 无折痕，反射光连续 |
| G2（曲率连续） | 边界曲率相同 | 曲率向量连续 | 反射光平滑过渡 |
| G3（曲率变化率连续） | 曲率变化率一致 | 曲率导数连续 | 极高质量的A级曲面 |

### 7.3 检查流程

```
1. 生成初始曲面
    ↓
2. 斑马纹检查 → 调整截面曲线
    ↓
3. 高光检查 → 调整曲面控制点
    ↓
4. 曲率梳检查 → 调整曲率分布
    ↓
5. 国标合规检查 → 修正超限参数
    ↓
6. 几何约束检查 → 验证硬点关系
    ↓
7. 完成A级曲面
```

### 8.2 语言参数输入

- "设计一辆紧凑型电动车，短前后悬"

**解析逻辑**：
```javascript
// 关键词识别
if (input.includes('轿跑') || input.includes('跑车')) → 运动型参数
if (input.includes('SUV') || input.includes('越野')) → SUV参数
if (input.includes('低重心') || input.includes('低矮')) → H *= 0.92
if (input.includes('流线型')) → FO *= 0.85
if (input.includes('豪华')) → L *= 1.05, WB *= 1.05

// 数值提取
if (input.match(/车长?([\d.]+)米?/)) → L = parseFloat(m[1])
if (input.match(/宽[\s:]*([\d.]+)/)) → W = parseFloat(m[1])
```

|------|------|
| 硬点（Hardpoint） | 决定车辆基本形态的关键几何控制点 |
| A级曲面（A-Class Surface） | 满足G2曲率连续、反射光平滑的高质量曲面 |
| Tumblehome | 乘员舱区域车顶向内倾斜的造型特征 |
| Lofted Surface | 通过截面曲线扫掠生成的曲面 |
| G0/G1/G2/G3 | 曲面连续性等级（位置/切线/曲率/曲率变化率） |
| H点 | 人体臀点，决定座椅位置和乘员空间 |

### C. 公式速查

|------|------|

| WC = GC + WR | 轮心高度 |
| fwx = FO + WR + 0.08 | 前轮心X坐标 |
| rwx = L - RO - WR - 0.08 | 后轮心X坐标 |
