# EVOLUTION AI 汽车造型开发算法总结

## 核心算法流程

### Phase 1: 硬点参数输入
```
输入参数 (14个):
- L: 车长 (m)
- W: 车宽 (m)
- H: 车高 (m)
- WB: 轴距 (m)
- FO: 前悬 (m)
- RO: 后悬 (m)
- GC: 离地间隙 (m)
- WR: 轮径 (m)
- TW: 轮距 (m)
- AA: A柱角 (°)
- CA: C柱角 (°)
- WL: 腰线高度偏移 (m)
- WBulge: 轮拱凸起 (m)
```

### Phase 2: 硬点坐标推导
```python
# 轮心坐标
fwx = FO                    # 前轮心X
rwx = L - RO                # 后轮心X
wcy = GC + WR               # 轮心高度
fwz = TW/2 + WR*0.35        # 轮心外侧偏移

# 车身高度关键点
noseTipY = GC + 0.43        # 前保险杠上沿
hoodY = GC + 0.66           # 发动机盖高度
waistY = GC + WL            # 腰线高度
if hoodY >= waistY: hoodY = waistY - 0.05  # 修正跑车

# A柱推导
aBaseX = fwx + 0.10         # A柱底在前轴后100mm
aTopY = H * 0.92            # A柱顶高度
aTopX = aBaseX + (aTopY - waistY) / tan(AA°)

# C柱推导
cBaseX = rwx + 0.30         # C柱底在后轴后300mm
cTopX = cBaseX - 0.50       # C柱顶在C柱底前500mm
cTopY = aTopY - 0.03

# 宽度关键点
frontFenderW = W/2 * 0.95   # 前翼子板最宽
cabinW = W/2 * 0.86         # 乘员舱宽度(tumblehome)
rearFenderW = W/2 * 0.95    # 后翼子板最宽
```

### Phase 3: 轮廓曲线生成
```
侧视轮廓曲线 (smoothstep插值):
- 车头 (t<0.08): 鼻端渐高至发动机盖
- 发动机盖 (0.08-0.20): 平滑过渡到腰线
- A柱 (0.20-0.40): 腰线→A柱顶
- 车顶 (0.40-0.60): 平直段
- C柱 (0.60-0.80): 车顶→C柱顶→腰线
- 车尾 (0.80-1.00): 平滑下降

顶视宽度曲线:
- 车头 (t<0.05): 窄→宽
- 前翼子板 (0.05-0.35): 最宽
- 乘员舱 (0.35-0.65): tumblehome收窄
- 后翼子板 (0.65-0.92): 最宽
- 车尾 (0.92-1.00): 收窄
```

### Phase 4: 截面扫掠
```
50站截面 × 24点 = 车身网格
每站截面 = 椭圆截面(底部Y→顶部Y) × 宽度变化 × 轮拱切口

轮拱切口公式:
if |z| > fenderW - sqrt(WR² - (y-wcy)²):
    切除该点
```

### Phase 5: 车身网格生成
```
三角面连接:
- 同一截面内: 相邻点连接
- 相邻截面间: 对应点连接
- 中心线闭合: 左右对称闭合
```

## 关键设计原则

1. **国标约束**: GB 1589-2016 外廓尺寸限值
2. **物理真实**: 轮心位置、离地间隙、轮径等必须符合物理约束
3. **比例协调**: 轴距/车长比、高宽比等必须在合理范围
4. **标杆对齐**: 模型参数与真实市场车型偏差<50mm
5. **跑车特殊处理**: 低矮车型自动修正发动机盖高度
