# EVOLUTION AI 五阶段开发流程图

## Mermaid 流程图（可直接在支持Mermaid的编辑器中渲染）

```mermaid
graph TD
    subgraph Phase1["Phase 1: 硬点定义"]
        P1_IN["输入<br/>轮心位置 / H点 / 人眼点<br/>A/C柱角度"] --> P1_CORE["核心推导<br/>前轮心X = FO + WR + 0.08<br/>后轮心X = L - RO - WR - 0.08<br/>轮心Y = GC + WR"]
        P1_CORE --> P1_OUT["输出<br/>硬点坐标矩阵"]
        P1_PARAMS["关联参数<br/>FO · RO · WB · GC<br/>WR · TW · AA · CA"] -.-> P1_CORE
    end

    subgraph Phase2["Phase 2: 三曲线系统"]
        P2_IN["输入<br/>硬点坐标 / 设计意图"] --> P2_CORE["三条骨架曲线<br/>① 侧视轮廓曲线 → Y坐标变化<br/>② 顶视宽度曲线 → Tumblehome内倾<br/>③ 截面变形函数 → 垂直轮廓"]
        P2_CORE --> P2_ALGO["smoothstep插值<br/>t²(3-2t) 确保G1连续"]
        P2_ALGO --> P2_OUT["输出<br/>车身骨架曲线"]
        P2_PARAMS["关联参数<br/>L · W · H · WL<br/>WBulge · AA · CA"] -.-> P2_CORE
    end

    subgraph Phase3["Phase 3: 参数化建模"]
        P3_IN["输入<br/>用户参数调整 / AI推荐"] --> P3_CORE["14个硬点参数实时驱动<br/>L→整体比例 | WB→前后悬分配<br/>FO→前鼻长度 | RO→车尾长度<br/>H→重心位置 | WL→侧面比例"]
        P3_CORE --> P3_FLOW["参数驱动流程<br/>调整参数 → 硬点推导<br/>→ 曲线生成 → 曲面扫掠 → 渲染"]
        P3_FLOW --> P3_OUT["输出<br/>更新后的车身曲面"]
        P3_PARAMS["关联参数<br/>L · W · H · WB<br/>FO · RO · WL"] -.-> P3_CORE
    end

    subgraph Phase4["Phase 4: A级曲面"]
        P4_IN["输入<br/>三曲线输出 / 截面数量"] --> P4_CORE["Lofted Surface截面扫掠<br/>50站截面 × 16层宽度<br/>生成G2连续闭合环面"]
        P4_CORE --> P4_CHECK["质量检查<br/>F5 斑马纹 → 光顺性<br/>F6 高光 → 反射连续性<br/>F7 曲率梳 → 曲率分布"]
        P4_CHECK --> P4_OUT["输出<br/>A级曲面"]
        P4_PARAMS["关联参数<br/>WBulge · AA · CA<br/>WR · TW"] -.-> P4_CORE
    end

    subgraph Phase5["Phase 5: AI辅助设计"]
        P5_IN["输入<br/>自然语言 / 设计需求"] --> P5_CORE["五大AI功能<br/>💬 语言参数输入<br/>🤖 智能参数推荐<br/>📈 造型趋势分析<br/>🚀 概念车探索<br/>🔄 多方案对比"]
        P5_CORE --> P5_OUT["输出<br/>参数方案 / 造型推荐<br/>趋势报告"]
    end

    Phase1 -->|"硬点坐标"| Phase2
    Phase2 -->|"骨架曲线"| Phase3
    Phase3 -->|"参数驱动"| Phase4
    Phase4 -->|"曲面质量"| Phase5
    Phase5 -->|"参数反馈"| Phase3
```
