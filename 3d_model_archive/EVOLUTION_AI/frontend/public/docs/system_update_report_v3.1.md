### 2.2 卡尔曼滤波参数平滑

#### 一维卡尔曼滤波器 (KalmanFilter1D)

**状态模型**: 位置-速度对 `x = [p, v]`，状态转移矩阵 `F = [[1,dt],[0,1]]`

**核心方程**:

```
预测:  x̂ₖ⁻ = F·x̂ₖ₋₁,  Pₖ⁻ = F·Pₖ₋₁·Fᵀ + Q
更新:  K = Pₖ⁻·Hᵀ / (H·Pₖ⁻·Hᵀ + R)
       x̂ₖ = x̂ₖ⁻ + K·(z - H·x̂ₖ⁻)
       Pₖ = (I - K·H)·Pₖ⁻
```

**调参指南**:

| 参数 | 默认值 | 作用 | 调优方向 |
|------|--------|------|----------|
| Q (过程噪声) | 0.001 | 信任模型预测 | Q↑ → 更跟随测量值 |
| R (测量噪声) | 0.01 | 信任测量值 | R↑ → 更平滑 |
| dt | 1.0 | 采样间隔 | 与采样密度匹配 |

**典型场景**: 侧视轮廓平滑 `Q=0.0003, R=0.002`；硬点去噪 `Q=0.001, R=0.01`

#### 多变量卡尔曼滤波器 (MultiVarKalmanFilter)

同时平滑 L, W, H, WB 等多个相关参数，保持参数间一致性。采用 Joseph 形式协方差更新 `P = (I-KH)·P·(I-KH)ᵀ + K·R·Kᵀ`，数值更稳定。

#### 便捷函数

```python
def kalman_smooth_profile(profile, Q=0.001, R=0.01):
    """对一维序列应用卡尔曼滤波平滑"""
    kf = KalmanFilter1D(Q=Q, R=R)
    return [kf.predict() or kf.update(val) or kf.x[0] for val in profile]
```

### 2.3 NURBS 曲面系统

#### B样条基函数 (de Boor 算法)

采用 Cox-de Boor 递推公式，二分查找 knot span 后逐阶递推。边界处理: `u≤knots[0]` 时 `N[0]=1`，`u≥knots[-1]` 时 `N[-1]=1`，分母 `|denom|<1e-15` 时置零。

#### 节点向量

Clamped 节点向量: 起止各 p+1 重节点，内部均匀分布，保证曲面通过首末控制点行。

#### NURBS 曲面求值

**数学定义**: `S(u,v) = ΣᵢΣⱼ wᵢⱼ·Nᵢ(u)·Nⱼ(v)·Pᵢⱼ / ΣᵢΣⱼ wᵢⱼ·Nᵢ(u)·Nⱼ(v)`

**优化实现**: 预计算基函数矩阵 `N_u_all`, `N_v_all`，避免重复计算。


```
|←── ho ──→|←──────── co ────────→|←── to ──→|
```

过渡宽度与轴距成比例 (`trans = WB * 0.05`)，使用 smoothstep 平滑过渡。

#### Tumblehome 截面

#### 控制网格截面生成

沿 v 方向分段 smoothstep 插值:

```

v ∈ [0.20, 0.45): 门槛 → 腰线
v ∈ [0.45, 0.75): 腰线 → 肩部
v ∈ [0.75, 1.00]: 肩部 → 车顶中心
```

右半截面镜像至左半截面 (Z坐标取反)。

#### 轮拱权重增强

轮拱区域 NURBS 权重增大: `arch_factor = 1.0 + 0.5 * (1 - dist / (WR * 2))`，仅影响侧面控制点。

### 2.5 曲面质量分析

使用有限差分法计算第一、第二基本形式，推导主曲率 k₁, k₂:

```
k₁ = (L·G - M·F) / (E·G - F²)
k₂ = (N·E - M·F) / (E·G - F²)
平均曲率 H = (k₁ + k₂) / 2
高斯曲率 K = k₁ · k₂
```

**参考基准** (BMW 3系级别): 平均曲率 0.01~0.05，标准差 <0.03，最大值 <0.5 (1/m)

| 症状 | 可能原因 | 解决方案 |
|------|----------|----------|
| 侧视轮廓锯齿 | 截面插值区间粗糙 | 增加 v 方向分段数或控制点数 |
| 车顶过低 | shoulderY→topY 过渡错误 | 检查 `shoulderY_c` 权重比例 |
| 车尾过度收缩 | tailTaper/noseTaper 耦合 | 分离前后锥度独立计算 |
| 轮拱处凹陷 | 权重增强不足 | 增大 `arch_factor` 系数 |

### 5.2 参数敏感性

| 参数 | 敏感度 | 影响范围 |
|------|--------|----------|
| shoulderW | 高 | 全车肩部宽度、车顶宽度 |
| CA (C柱角) | 高 | 后风挡斜度、行李箱高度 |
| archBulge | 中 | 轮拱区域外凸量 |
| RA (Raked角) | 中 | 车顶后掠程度 |
| Q/R (卡尔曼) | 低 | 参数平滑程度 |

### 5.3 性能参考

|----------|--------|------|----------|
| 20×32 | 80×64 | ~4.1s | 标准质量 |
| 30×48 | 120×96 | ~15s | 高精度 |
| 15×24 | 40×32 | ~0.5s | 3D打印预览 |
