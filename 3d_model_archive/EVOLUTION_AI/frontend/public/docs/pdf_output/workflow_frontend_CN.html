2.3 参数分组与约束
尺寸组(9) 决定车辆外廓与空间布局，是deriveHardpoints()的主输入。
造型组(6) 控制车身曲面特征（柱角、腰线、肩部），影响sideProfile()与topWidthProfile()。
高级组(6) 微调局部造型细节（引擎盖斜率、Tumblehome、后部收缩），作用于曲面的局部变形。

硬点推导规则：
frontAxleX = FO - L/2
rearAxleX = L/2 - RO
hoodHeight = GC + WR + hoodSlope×FO
roofPeak = H - GC
beltLineY = WL
deckLidY = GC + WR×deckRatio
共推导 48 个二级硬点，耗时 <0.1ms。
